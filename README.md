# 🏛️ Museo Virtual Interactivo — Revoluciones del Siglo XVIII

> **Actividad de compromiso de aprendizaje** | Ciencias Sociales · Grado 8  
> Institución Educativa Gabo · Cartago, Valle del Cauca, Colombia

---

## 📖 Descripción

Aplicación web educativa interactiva que simula un museo virtual sobre las grandes revoluciones del siglo XVIII: la Revolución Francesa, la Revolución Americana y la Revolución Industrial. Los estudiantes exploran salas temáticas, responden evaluaciones y obtienen un **Diploma de Curador Experto** al completar el recorrido.

### ✨ Características principales

- 🎨 Interfaz glassmorphism con partículas animadas
- 🗺️ Salas temáticas con galería de imágenes interactiva
- 📝 Evaluaciones con retroalimentación inmediata
- 🏆 Sistema de puntuación y logros
- 📄 Generación de diploma en PDF personalizado
- 📱 Diseño responsive para móviles y escritorio

## 🚀 Uso

1. Clona o descarga este repositorio
2. Asegúrate de que la carpeta `imagenes/` esté en la misma ubicación que `index.html`
3. Abre `index.html` en cualquier navegador moderno (Chrome, Firefox, Edge)
4. No requiere instalación ni conexión a internet (excepto para íconos de Font Awesome)
